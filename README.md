# 🚀 Neon Pong -- Flutter Arcade Game

A fast-paced, neon-styled 2D Pong game built using **Flutter** with
custom rendering, real-time physics, and AI-based gameplay.

------------------------------------------------------------------------

## 🎮 Features

### 🕹 Game Modes

-   Machine vs Man (AI Mode)
-   The Void Wall (Endless Mode)

### ⚙ Difficulty Levels

-   Easy
-   Medium
-   Hard

### 🧠 Smart AI

-   Reaction delay
-   Dynamic speed scaling
-   Fair but challenging

### 🎨 Custom Rendering

-   Built using CustomPainter & Canvas
-   Fully manual drawing (no game engine)

### 🌌 Dynamic Background

-   Procedural stars
-   Flickering animation
-   Gradient-based themes

### ⚡ Game Loop

-   60 FPS using Timer
-   Real-time updates

### 💥 Physics

-   Paddle collision
-   Wall collision
-   Speed scaling with limits

### 🏆 Scoring

-   XP system
-   Round-based wins

------------------------------------------------------------------------
